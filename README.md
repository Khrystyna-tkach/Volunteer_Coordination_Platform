🤝 Платформа координації волонтерської допомоги HelpBridge
Веб-додаток, створений для швидкої взаємодії між людьми, що потребують допомоги, та волонтерами. Система дозволяє збирати заявки, розподіляти їх та відстежувати прогрес виконання в режимі реального часу.

🚀 Основні можливості
Платформа розділена на три рівні доступу:

👤 Користувач (Отримувач): створення заявок на допомогу, опис проблем та відстеження статусів.

🛡️ Волонтер: перегляд списку актуальних заявок, взяття їх у роботу та зміна статусів.

🔑 Адміністратор: повний контроль над системою, модерація користувачів та заявок.

🛠 Технологічний стек
Backend: Django Framework (Python)

Database: PostgreSQL

Frontend: HTML5, CSS3, JS (з акцентом на зручність UI/UX)

Design: Figma

📋 Функціональні вимоги (SRS)
Система реалізує наступний життєвий цикл заявки:

Створення: Користувач заповнює форму (ім'я, контакти, опис проблеми, адреса).

Обробка: Заявка отримує статус "Нова". Волонтер переглядає список і бере її в роботу (статус "В процесі").

Завершення: Після надання допомоги статус змінюється на "Виконана".

Примітка: Доступ до функцій суворо обмежений ролями користувачів для забезпечення безпеки даних.
